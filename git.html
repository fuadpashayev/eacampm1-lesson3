Steps

  1. google -> download git
  2. create github profile
  3. create github repository
  4. open terminal in vscode
  5. execute commands in github new repository page (exclude: readme.md)
  6. send changes commands:
    git add .
    git commit -m "logical and meaningful commit message"
    git push origin branch-name
  7. create new branch command:
    git checkout -b meaningul-branch-name
  8. list all branches:
    git branch
  9. change branch:
    git checkout branch-name
  10. pull new codes:
    git pull origin branch-name
